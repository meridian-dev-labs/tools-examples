## Key Selection Criteria

### Fee Structure
- **Maker fees** typically 0.05%–0.2% (lower = better for frequent traders)
- **Taker fees** typically 0.1%–0.25% (charged when taking existing orders)
- **Discount tiers** available via holding native tokens (BNB, OKB, HT)

### Registration Process
- Email verification (all)
- Phone/2FA (recommended)
- KYC Level 1: limited withdrawal
- KYC Level 2: full access (identity document required)

### Security Considerations
- **2FA enforcement**: Required or optional
- **Cold storage**: Percentage of user funds kept offline
- **Insurance funds**: Protection against exchange insolvency
- **Regulatory licenses**: Operating jurisdictions

## Recommended Decision Matrix

| Use Case | Best Exchange | Reason |
